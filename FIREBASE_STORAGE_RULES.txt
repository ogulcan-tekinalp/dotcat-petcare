Firebase Storage Kuralları
==========================

Firebase Console'da Storage > Rules sekmesine git ve şu kuralları yapıştır:

rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /users/{userId}/{allPaths=**} {
      allow read, write: if request.auth != null && request.auth.uid == userId;
    }
  }
}

Bu kurallar:
- Sadece giriş yapmış kullanıcıların kendi dosyalarını okuyup yazmasına izin verir
- Her kullanıcı sadece kendi klasörüne (users/{userId}/...) erişebilir
- Başka kullanıcıların dosyalarına erişim yok

Kuralları değiştirdikten sonra "Publish" butonuna tıkla!
